# titanic-survival-prediction
Kaggle getting started competition

## Requirements
### Jupyter notebooks
* docker
* docker-compose

## scripts
* libomp (for Mac)

```shell
brew install libomp
```

## Usage
### Jupyternotebook
1. Launch Jupyter
```shell
$ docker-compose up
```

2. Access the UI link `localhost:8888` with a token

### scripts
1. Install dependencies library
```
pip install -r requirements.txt
```
